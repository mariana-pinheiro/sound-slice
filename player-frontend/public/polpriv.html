Política de Privacidade

A tua privacidade é importante para nós. Esta política explica como recolhemos, usamos e protegemos as tuas informações na plataforma SoundSlice.

1. Recolha de Dados

Podemos recolher dados como nome, email, dados de pagamento e interações na plataforma.

2. Utilização

Usamos os teus dados para fornecer os serviços da SoundSlice, gerir contratos inteligentes, processar pagamentos e melhorar a experiência do utilizador.

3. Partilha

Não vendemos os teus dados. Podemos partilhá-los apenas com parceiros de confiança necessários para o funcionamento da plataforma.

4. Segurança

Usamos tecnologias modernas para proteger a tua informação contra acessos não autorizados.

5. Direitos

Tens o direito de aceder, corrigir ou eliminar os teus dados pessoais. Contacta-nos em privacidade@soundslice.pt.
